# Changelog

All notable changes to **StyleX Fold** will be documented in this file.

## [1.0.4] - 2026-01-02

Initial release.

### Commands

- `StyleX: Fold to Names` — Collapse style bodies to show only names (`item: { ... }`)
- `StyleX: Fold Entire Blocks` — Collapse entire `stylex.create`/`keyframes` blocks
- `StyleX: Unfold to Names` — Expand blocks but keep bodies folded
- `StyleX: Unfold All` — Expand everything completely

### Supported Patterns

- `stylex.create({ ... })`
- `stylex.keyframes({ ... })`
